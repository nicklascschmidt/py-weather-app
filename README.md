# py-weather-app
Python app to search weather by city

## Basic Info
User searches for weather by city and adds/removes cities from the UI, which is populated from a database.

## Technologies
* Flask (render_template, request, redirect)
* Flask-SQLAlchemy
* OpenWeatherMap API
* Bulma CSS framework
* HTML
